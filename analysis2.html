Data Dictionary

Data is in .xlsx format

1. Customer Information

Column Headers Data Type Description
Customer ID Text Unique customer identifier
Age Numeric Age of the customer
Gender Text Gender of the customer
Preferred Item Text The item preferred by the customer

2. Employee Information

Column Headers Data Type Description
Employee ID Text Unique identifier for each employee
Employee Name Text The names of the cafe's employees
Role Text The roles of the employees
Location Text The location of the cafe's employees
Sales Target Numeric The total sales target assigned by each employee
Sales Achieved Numeric The total sales achieved by each employee

3. Stock Information

Column Headers Data Type Description
Stock Item Text The name of each stock item
Current Stock Numeric The current stock level of each item
Ideal Stock Numeric The ideal stock level of each item

4. Feedback Information

Column Headers Data Type Description
Feedback ID Text Unique identifier for each feedback
Customer ID Text Unique identifier for each customer
Sentiment Text The customer sentiment on the item
Comment Text People's review of each item
Rating Numeric The numerical rating of each item

5. Sales Information

Column Headers Data Type Description
Sales ID Text Unique identifier for each sale
Customer ID Text Unique identifier for the customer who made the purchase
Product Name Text The name of the items the cafe sells
Quantity Sold Numeric The quantity of the items sold in the transaction
Price per Unit Sold Numeric The price of each item sold in the transaction
Location Text The location of the cafe's outlet
Sales Date Date The date on which the sale occurred
Total Sales Numeric The total price of the items sold in the transaction

Key Indicators

  • Sales Summary: Over 1,000 transactions and 3,000+ units sold generated approximately $12.6K in revenue, with an average customer rating of 3.05, highlighting steady sales performance and room for improved customer satisfaction.

Sales Movement

  • Across the 3 years sales trend, there is always a deep in September.
  • February is always a pickup month.

Productivity

  • The most productive year was 2024.
  • Sales was relatively low in the first half of 2025.
  • Average sales kept dropping across the years.
